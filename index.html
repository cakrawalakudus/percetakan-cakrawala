Hubungkan Ke Google Sheets Asli

Ikuti langkah simpel ini untuk mengirim data dari aplikasi digital ke file Google Sheet milik sekolah Anda secara otomatis.

Jika diisi, setiap kali menekan tombol "Simpan Ke Laporan Sheet", data langsung masuk ke Google Sheet Anda!

Langkah Pemasangan Code Di Google Sheet:

  1. Buka Google Sheet baru milik sekolah Anda.
  2. Klik menu Extensions / Ekstensi → Apps Script.
  3. Hapus semua kode bawaan, lalu tempelkan (paste) kodeScript di bawah ini.
  4. Klik Deploy / Terapkan → New deployment / Terapkan baru.
  5. Pilih Jenis: Web app / Aplikasi Web. Set "Who has access / Siapa yang memiliki akses" menjadi Anyone / Siapa saja.
  6. Salin URL Web app lalu tempelkan di kolom input URL di atas. Selesai!
